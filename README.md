### Project's Goal:

This project aimed to demonstrate our understanding of game development principles, including 2.5D graphics, physics simulation, and UI design, while also gaining experience in collaborative software development using Git and agile methodologies.

### How to play?

Long story short, the project was inspired by old game called Lamborghini American Challenge.
The game mechanics are similar, but the implementation differs a lot!

Players gain points based on distance they rode. However, there are some obstacle which players have to pass or the game will be over.
If players manage to pass obstacle close enough they gain extra points.

![](https://imgur.com/TbU2o3Z.png)

The game saves players' scores only if they logged in. Otherwise, the score will be demolished.


![](https://imgur.com/yIB9Yuf.png)

### Key Features:

- Simulation of 2.5D environment using Parallax effect
- Dynamic obstacle generation
- Dynamic road generation
- Simple physics for cars' movement
- Custom Retro UI
- Sprite animation
- Unit tests

### Tools & Technologies:

- Python: programming language
- Pygame lib: a base for implementing a game
- Pytest lib: simple lib for creating easy unit test
- Aseprite: for creating game sprites
